# Slime

 * Android用のIMEです。
 * 以下のような特徴があります。
  * モードが存在しない
  * T9式子音入力が可能
  * スライドまたはマルチタッチにより50音入力が可能
  * 英数字入力が簡単
  * 候補が無いとググる

## インストール

 * AntまたはEclipseでビルドする
 * Slime.apkをAndroidにインストールする
 * 「設定」→「言語とキーボード」で「Slime」を選択
 * 入力システム利用許可にOKする
 * 入力フィールドで長押しして入力メソッドを選択する
